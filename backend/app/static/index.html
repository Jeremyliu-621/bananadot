use in godot
  1. Unzip the file.
    You should get a folder named bananadot_button.
  2. Move that folder into your Godot 4 project's root.
    It should sit next to your project.godot file. Godot will show it at res://bananadot_button/.
  3. In Godot, open bananadot_button/example.tscn.
    Run the scene (F6) — hover and click the button to see the states work.
  4. To use it in your own scenes, either instance example.tscn as a PackedScene, or add a TextureButton node and wire its texture properties to the PNGs in assets/.
      Full notes are in the README.md inside the folder.
